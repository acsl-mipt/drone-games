Установка зависимостей:
./install.sh

Запуск с судейством:
./start.sh ФАЙЛ_НАСТРОЕК

Примеры файла настроек: example_f.rc и example_r.rc для синхронного полета и гонок соответственно. 
Необходимо прописать количество аппаратов, строку запуска управляющего скрипта, название своей команды.

Выдача работает корректно только с мирами formation.world, formation\_2.world, race.world и race\_2.world.
Отображение текущего задания в судействе работает корректно только с formation\_2.world и race\_2.world (подсчет будет работать и с formation.world и race.world).
